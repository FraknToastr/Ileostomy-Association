IASA static mock-up

Root-level files only:
- index.html
- styles.css
- script.js
- SVG asset files

Open index.html in a browser.

Notes:
- The mock-up uses basic JavaScript for believable front-end behaviour only.
- No orders, payments, contact messages, donations, translations, or data are submitted.
- The Important Dates modal loads the supplied calendar URL in an iframe:
  https://frakntoastr.github.io/Ileostomy-Association/calendar/index-v2.html
- All assets are local SVG files except the calendar iframe.
