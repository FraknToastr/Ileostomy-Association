ABOUT

Stoma Association Finder

This browser-based map links Australian postcode locality points to the nearest Stoma Association using the ArcGIS Pro Near results stored in NEAR_FID and NEAR_DIST.

Mapping: MapLibre GL JS. Basemaps: OpenStreetMap and Esri World Imagery. Postcode and association data are embedded locally in the project.
